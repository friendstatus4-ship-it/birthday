Put your photos in this folder with these exact names and they'll appear automatically:

Timeline photos:
  memory-01.jpg ... memory-09.jpg

Gallery photos:
  gallery-01.jpg ... gallery-06.jpg

Until a file exists, a soft placeholder is shown automatically, so nothing
breaks in the meantime. You can add more/fewer gallery photos too — just
add or remove entries in the CONFIG.gallery list near the bottom of
index.html (search for "CONFIG").
